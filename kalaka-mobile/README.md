# React Native Starter

This boilerplate project is _super_ helpful to kick-start your next project, as it provides a lot of the common tools you may reach for, all ready to go. Specifically:

- **Expo**
- **TypeScript**
- **Redux**
- **React Navigation**
- **Native Base**
- **i18n**
